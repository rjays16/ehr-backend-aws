Tools Menu
 Study Browser

The study browser is used to select and/or change the series being viewed. This feature is enabled by clicking the study browser icon  located in the tools menu.  Enter search criteria and click the search button to see a list of matching studies.  Select the appropriate study and series for viewing.


*While the Study Browser is open all features and tools are disabled
 Split Screen

The NovaPacs Lite Viewer allows for comparison of series within the same study or from different studies. This feature is enabled by clicking the split screen icon  located in the tools menu. This launches the study browser prompting the selection of a second study/series. The series 1 will be on the left and series 2 on the right. To exit or disable the split screen feature click the icon located in the tools menu of either series.


 Reports

The NovaPacs Lite Viewer allows for the viewing of diagnostic reports. This feature is enabled by clicking the reports icon  located in the tools menu. The report will be opened in a new window.


 Help

Help is available on any of the features or tools by clicking the help icon located in the tools menu.


 Series Navigation

To navigate to other series in the same study, simply select the thumbnail image corresponding with the series on the thumbnail list at the right side of the screen.


 LayOut Options

The NovaPacs Lite Viewer supports various image layout options.

The LayOut may be changed by clicking the LayOut Options icon  located in the tools menu.  A new selectable menu will appear below the LayOut options icon. From the new menu select the desired layout.

 Measuring

The NovaPacs Lite Viewer allows the users to perform measuring on the Images. This feature can be enabled by selecting the Measure Icon  located in the tools menu.  With the measure feature turned on, hold the left mouse button down and drag the desired distance to be measured.

 Magnification
The NovaPacs Lite Viewer allows users a magnification option.

This tool may be enabled by clicking the Magnification Option Icon   located in the tool menu. With the Magnification Option enabled move the mouse over the images.  


 Window Level Options

The NovaPacs Lite Viewer allows the users to change the Window & Level of images. This feature can be enabled by by clicking the right mouse button and moving the mouse pointer directly above the Window Level option displayed. A new selectable menu will appear to the right with preset window level options based on the modality type of the series. By selecting one of the preset options the window level will be changed.


*Changing the window level only affects images being viewed in the Lite Viewer. It does not modify the actual images.
 Tools & Features

The Features and Tools menu can be accessed at any time by selecting the Help Icon.

 Tools Menu

The Tools menu  is located at the top of the browser.
